# tele v1.0.1

This patch release corrects the release runbook after repository-level release
immutability was enabled. Release assets published from v1.0.1 onward are
enforced immutable by GitHub; older releases remain append-only by project
policy and must never be replaced or retagged.

The CLI contract, `tele/v1` machine schema, and platform support are unchanged
from v1.0.0.
